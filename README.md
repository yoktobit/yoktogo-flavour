# yoktogo-flavour
This command adds flavours to a project, e.g. to add cue support. Say... partial archetypes.

# Installation
go install github.com/yoktobit/yoktogo-flavour@latest

# Usage
ygf get <RepoName>

# Example
ygf get github.com/yoktobit/yoktogo-flavour-cue
